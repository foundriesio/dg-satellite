# Contributing to Device Gateway Satellite

The project is confidential and proprietary.

1. New files need proper marking at top:
   ```
   Copyright (c) Qualcomm Technologies, Inc. and/or its subsidiaries.
   All rights reserved.
   Confidential and Proprietary - Qualcomm Technologies, Inc.
   ```

1. Commits will be signed with `@foundries.io` emails.

1. Dependencies must be permissively licensed. Licenses like AGPL and GPLv3
   are not permitted.
